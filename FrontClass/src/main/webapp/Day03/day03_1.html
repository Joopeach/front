head 태그 안에서 style 선언

선택자 : 꾸미기 대상

1. 태그명 { 속성 = "속성값"; }

2. .클래스명 { 속성 = "속성값"; }

3. #id명 { 속성 = "속성값"; }

우선순위 : id>클래스>태그

글꼴관련 속성 1. font-family = "글꼴명" 2. color = // 1. 색 이름, 헥스값, RGB [http://htmlcolorcodes.com/color-picker] background-color : 배경색 3. text-decoration : 밑줄 4. text-align : 텍스트 정렬 5. text-shadow : x축, y축, 흐름정도, 색상 6. font-size : 30px; /*글자크기*/ font-style : italic; /*기울기*/ font-weight : bold; /*글자 굵게 진하기 */ line-height : /*줄간격*/ letter-spacing : 10px; /*글자간격 : 자간*/ word-spacing : 10px; /*단어간격 : 어간*/
